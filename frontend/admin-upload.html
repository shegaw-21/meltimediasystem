File Guidelines

  • 🎬 Video: MP4 format only. H.264 encoding recommended for best browser compatibility.
  • 🎵 Audio: MP3 format only. This is the summary audio displayed alongside the video.
  • 📄 Transcript: Upload a .docx for best results. The server auto-converts it with estimated timestamps. JSON is also accepted.

Part Naming

Use a consistent scheme like part16, part17, etc. Uploading a file with an existing part name will replace that content.
